The Graphical User Interface (GUI) can be obtained from GitHub. To run the offline version of the program, please install R from http://r-project.org and enter the following commands at the prompt: 
install.packages('devtools')
install.packages('shiny')
library(devtools)
install_github('pvermees/IsoplotR')
install_github('pvermees/IsoplotRgui')

Once all these packages have been installed, you can run the browser-based graphical user interface by typing: library(IsoplotRgui) IsoplotR()
