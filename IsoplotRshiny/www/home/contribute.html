Users can contribute to IsoplotR's development in a number of ways:
  1. Any feedback and bug reports are gratefully accepted. If you encounter any problems, then please send a reproducible example in a .csv (if you use the command line) or .json (if you use the GUI) format to p.vermeesch [at] ucl.ac.uk and I will get back to you asap.
  2. If you know how to program in R, then feel free to fork Isoplot's source code from GitHub and modify or recycle it. Submit a pull request if you have fixed a bug.
  3. Host a local mirror version of the online GUI. This will increase the responsiveness of the site outside the United Kingdom. Installation instructions are provided on GitHub.
  4. IsoplotR is and always will be free to use. However, the development of the code and maintenance of the server do take time and money. If you would like to make a financial contribution, then please click the button below. Cheers!
